# Настройка уведомлений в Telegram

Этот гайд поможет настроить уведомления о новых пользователях и сообщениях в Telegram.

## Шаг 1: Создание Telegram бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Введите имя для вашего бота (например, "Portfolio Notifications")
4. Введите username для бота (должен заканчиваться на "bot", например, "portfolio_notify_bot")
5. BotFather отправит вам токен бота - **сохраните его!**
   
   Пример токена: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

## Шаг 2: Получение Chat ID

1. Найдите бота **@userinfobot** в Telegram
2. Отправьте ему любое сообщение
3. Бот ответит вашим Chat ID - **сохраните его!**
   
   Пример Chat ID: `123456789`

## Шаг 3: Настройка проекта

1. Создайте файл `.env` в корне проекта (если его нет)
2. Добавьте следующие строки:

```env
TELEGRAM_BOT_TOKEN=ваш-токен-бота
TELEGRAM_ADMIN_CHAT_ID=ваш-chat-id
```

Пример:
```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_ADMIN_CHAT_ID=123456789
```

## Шаг 4: Перезапуск сервера

1. Остановите сервер (Ctrl+C в терминале)
2. Запустите снова: `node backend/server.js`

## Проверка работы

### Тест уведомления о регистрации:
1. Зарегистрируйте нового пользователя на сайте
2. Вы должны получить сообщение в Telegram:
   ```
   🆕 Новый пользователь зарегистрирован!
   
   👤 Имя: Иван Иванов
   📧 Email: ivan@example.com
   🕐 Время: 27.02.2026, 15:30:45
   ```

### Тест уведомления о сообщении:
1. Войдите как обычный пользователь (не админ)
2. Отправьте сообщение в чат
3. Вы должны получить сообщение в Telegram:
   ```
   💬 Новое сообщение!
   
   👤 От: Иван Иванов
   📧 Email: ivan@example.com
   📝 Сообщение:
   Привет! Это тестовое сообщение.
   
   🕐 Время: 27.02.2026, 15:35:20
   ```

## Отключение уведомлений

Если вы хотите временно отключить уведомления:
1. Удалите или закомментируйте строки с `TELEGRAM_BOT_TOKEN` и `TELEGRAM_ADMIN_CHAT_ID` в файле `.env`
2. Перезапустите сервер

## Troubleshooting

### Уведомления не приходят:
- Проверьте правильность токена и Chat ID
- Убедитесь, что файл `.env` находится в корне проекта
- Проверьте консоль сервера на наличие ошибок
- Убедитесь, что вы отправили хотя бы одно сообщение вашему боту (нажмите "Start")

### Ошибка "Chat not found":
- Отправьте команду `/start` вашему боту в Telegram
- Убедитесь, что Chat ID правильный

### Ошибка "Unauthorized":
- Проверьте правильность токена бота
- Убедитесь, что токен не содержит лишних пробелов

## Дополнительные возможности

Вы можете настроить бота для отправки уведомлений в группу:
1. Создайте группу в Telegram
2. Добавьте вашего бота в группу
3. Сделайте бота администратором группы
4. Используйте Chat ID группы вместо личного Chat ID

Для получения Chat ID группы:
1. Добавьте бота **@getidsbot** в группу
2. Отправьте команду `/start@getidsbot`
3. Бот отправит Chat ID группы (будет начинаться с минуса, например: `-123456789`)
